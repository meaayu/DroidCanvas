# Contributing to DroidCanvas

Thanks for considering a contribution! 🎨

## Getting Started

```bash
git clone https://github.com/<your-username>/DroidCanvas.git
cd DroidCanvas
./gradlew assembleDebug
```

Requirements: Android Studio, Kotlin 1.9+, API 26+.

## Workflow

1. Fork the repo and create a branch: `git checkout -b feature/amazing-feature`
2. Make your changes
3. Commit using [Conventional Commits](https://www.conventionalcommits.org/):

   | Type | Use for |
   |------|---------|
   | `feat:` | New feature |
   | `fix:` | Bug fix |
   | `chore:` | Maintenance, no behavior change |
   | `docs:` | Documentation only |

   Example: `git commit -m "feat: add image expand tool"`

4. Push and open a Pull Request
